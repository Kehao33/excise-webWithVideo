2.概述

jQuery 是一个简洁快速的 JavaScript 脚本库，它能让你在网页上简单的操作文档、处理事件、运行动画效果或者添加异步交互。jQuery 的设计会改变你写JavaScript 代码的方式，提高我们的编程效率。

5.JQuery 的函数工厂

JQuery 的函数工厂jQuery 的工厂函数“$”。无论我们使用哪种类型的选择符都需要从一个“$”符号和一对“()”开始。在“()”中通常使用字符串参数，参数中可以包含任何 CSS 选择符表达式。
